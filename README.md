# Peach-Printer2.0 Web打印模板设计器

#### 一、Peach Printer是什么？
Peach-Printer是一套 Web打印服务解决方，有独立的web打印工具，能通过简单的拖拉拽的方式设计打印模版，通过ajax post调用后台打印控件实现预览打印等功能，具有分页、汇总等功能；

#### 二、Peach Printer的功能特征
- **可视化模板设计器**: 提供了所见即所得的在线模板设计器，方便用户快速设计自定义报表，无需编写 HTML 代码。
- **动态修改模板内容**: 生成的模板数据采用 JSON 格式，可以动态修改模板内容，灵活方便。
- **便捷调用**: 集成简单，只需几行调用代码，无需引入 JS 文件。
- **简单易用**: 入门门槛低，非软件开发人员也能快速上手。
- **静默打印**: 支持静默打印（不预览直接打印），提高打印效率。
- **表格分页汇总**: 可以对表格进行分页打印，并自动添加汇总信息。
- **批量打印**: 支持批量打印多个文档，节省时间和精力。
- **导出PDF**: 可以将网页内容导出为 PDF 文件，方便保存和分享。
- **兼容性强**: 兼容主流浏览器。
- **免费使用（带水印）**: 可以免费使用，但打印的文档会带有水印。

#### 三、Peach Printer操作指南
Peach Printer 提供了在线和离线两种方式使用设计器：
##### 1. 在线使用设计器:
- 访问 Peach Printer 官网的在线设计器： peach-printer.netlify.app/
- 在在线设计器中设计并编辑你的打印模板。
- 完成设计后，下载模板文件（JSON格式）。
- 将下载的模板文件导入到你的业务系统中使用。
##### 2. 离线使用设计器:
- 在 Peach Printer 官网的“下载中心”下载设计器代码包。
- 将代码包部署到本地局域网 Web 服务器中。
- 访问本地 Web 服务器上的设计器，进行模板设计和编辑。
- 定期关注官网，更新到最新版本的设计器。
##### 3. 安装本地后台打印服务:
- 在 Peach Printer 官网的“下载中心”下载后台打印服务的安装包。
- 根据安装包的说明进行安装和配置。

#### 五、Peach Printer使用场景
- **电商平台**: 打印订单、发货单、快递单等。
- **企业管理系统**: 打印报表、合同、文档等。
- **医疗系统**: 打印病历、处方、检查报告等。
- **教育系统**: 打印试卷、成绩单、证书等。
- **各种需要打印网页内容的场景**: 例如打印网页文章、图片、表格等。尤其适用于需要非技术人员自定义打印模板的场景。

#### 六、Peach Printer运作模式
Peach Printer 的 Web 版模板编辑器允许用户可视化地设计打印模板，并生成 JSON 格式的模板数据。后台打印服务接收 JSON 数据和打印内容，将其渲染成最终的打印结果，并发送到打印机进行打印。
